## 🚀 How to Use

1. **Open the website:** Double-click `index.html` or use `Start-Process "index.html"` in PowerShell
2. **Start editing:** Click "Edit Mode" button to enable editing
3. **Customize content:** Click on any text to edit it directly
4. **Add/remove items:** Use the + Add buttons or trash icons
5. **Upload photo:** Click on the profile image placeholder
6. **Save changes:** Use Save button or Ctrl+S
7. **Print resume:** Use Print button for physical copies
